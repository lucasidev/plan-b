# EPIC-01: Catálogo público y exploración

**Status**: Not started
**BCs involved**: Academic (lecturas), Reviews (lecturas anonimizadas)

## Capability

Visitor anónimo puede explorar el catálogo (universidades, carreras, planes, materias, docentes), ver reseñas agregadas, y buscar por texto libre. Es el funnel de entrada al producto: antes de registrarse, el visitor evalúa la plataforma navegando.

## User Stories

- [US-001](../user-stories/US-001.md): Explorar catálogo de universidades y carreras
- [US-002](../user-stories/US-002.md): Ver materia con sus reseñas
- [US-003](../user-stories/US-003.md): Ver docente con sus reseñas
- [US-004](../user-stories/US-004.md): Buscar materia o docente por texto libre
