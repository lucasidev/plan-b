## Como autor de una reseña, quiero editar mi reseña para corregir o ampliar

Como autor de una Review en status='published', quiero un endpoint PATCH que re-corra el filter de contenido sobre el nuevo texto, marque "editada después de respuesta" si tenía TeacherResponse, y deje audit log.

## Acceptance Criteria

- [ ] `PATCH /api/me/reviews/{id}`.
- [ ] Solo desde status='published' (ADR-0012).
- [ ] Re-corre filter sobre nuevo texto.
- [ ] Si tenía TeacherResponse: marca "editada después de respuesta".
- [ ] Emite `ReviewEdited` + entry en audit log.

## Sub-tasks

- [ ] Comando `UpdateReviewCommand` + handler
- [ ] Endpoint Carter PATCH
- [ ] Re-run filter de contenido
- [ ] Audit log entry
- [ ] UI edit form
