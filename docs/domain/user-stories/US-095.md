[US-068](US-068.md) cubre el disable y [US-086](US-086.md) el audit del detalle, pero el listado en sí no tiene story. El mockup `AdmUsuariosList` del canvas no tiene story. Esta US arma la lista filtrable con link al detalle.

## Acceptance Criteria

- [ ] Link al detalle de usuario (donde viven el disable de US-068 y el audit de US-086).
