## Como member con email verificado, quiero crear un StudentProfile asociando carrera + plan + año de ingreso para desbloquear capabilities de alumno

Como member con email verificado, quiero crear un StudentProfile (CareerPlan + enrollmentYear) para desbloquear capabilities de alumno: cargar historial, simular, reseñar.

## Acceptance Criteria

- [ ] `POST /api/me/student-profiles` con `{ careerPlanId, enrollmentYear }`.
- [ ] Requiere User authenticated + email_verified_at NOT NULL.
- [ ] Valida CareerPlan existe.
- [ ] Valida no haya StudentProfile activo del mismo user para el mismo plan.
- [ ] enrollmentYear ∈ [1950, año actual].
- [ ] Crea StudentProfile con status=active.
- [ ] Retorna 201.
- [ ] Slice D del epic.

## Sub-tasks

- [ ] Comando `CreateStudentProfileCommand` + handler
- [ ] Endpoint Carter
- [ ] Validaciones (autenticado, email verificado, CareerPlan exists, sin duplicado)
- [ ] Integration tests
