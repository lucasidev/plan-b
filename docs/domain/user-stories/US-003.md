## Como visitante anónimo, quiero ver un docente con las reseñas donde aparece como docente reseñado para tener contexto al elegir comisión

Como visitante anónimo, quiero la página de un docente con metadata, agregados y reseñas donde figura como docente_reseñado, para usar el dato al elegir comisión.

## Acceptance Criteria

- [ ] `GET /api/teachers/{id}` retorna metadata + agregados.
- [ ] `GET /api/teachers/{id}/reviews` lista de reviews donde fue el docente reseñado, anonimizadas.

## Sub-tasks

- [ ] Read model agregados por teacher
- [ ] Endpoint Carter
- [ ] Página frontend
