## Como alumno con StudentProfile, quiero navegar simulaciones publicadas por otros alumnos del mismo plan + cuatrimestre para ver qué combinaciones se están considerando

Como alumno, quiero un endpoint que liste simulaciones shared del mismo plan + cuatrimestre (anonimizadas) con composición + métricas agregadas, para tener inspiración antes de armar la mía.
