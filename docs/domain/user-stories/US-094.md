## Como admin, quiero un flujo guiado de 5 pasos (Universidad → Carrera → Plan → Catálogo inicial → Listo) para dar de alta una universidad nueva de punta a punta

US-060 / US-061 / US-062 cubren las entidades sueltas (University, Career, Subject), pero no el flujo guiado end-to-end. Los mockups `AdmOnb*` del canvas no tienen story. Esta US arma el wizard que orquesta el alta completa de una universidad nueva.

## Acceptance Criteria

- [ ] Reusa los CRUD existentes (US-060/061/062) como pasos del wizard.
