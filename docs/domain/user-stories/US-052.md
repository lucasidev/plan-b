## Como moderator, quiero restaurar una reseña removida ante apelación del autor

Como moderator, quiero un endpoint que ponga una review removed de vuelta en published con justification; los reports upheld históricos NO se revierten (ADR-0011).

## Acceptance Criteria

- [ ] `POST /api/admin/reviews/{id}/restore` con `{ justification }`.
- [ ] Setea Review.status=published.
- [ ] Reports upheld históricos NO se revierten (ADR-0011).
- [ ] Audit log entry con action='restored'.

## Sub-tasks

- [ ] Comando `RestoreReviewCommand`
- [ ] Endpoint Carter
- [ ] Audit log
- [ ] UI restore button
