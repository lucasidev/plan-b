## Como admin, quiero un CRUD de comisiones (oferta de materia + cuatrimestre con docentes asignados) para que la cursada sea reseñable

Hoy no existe CRUD de comisiones en el backend (solo se seedean). El mockup `AdmComisiones` del canvas no tiene story. Esta US cierra ese gap: gestionar comisiones desde el backoffice para que cada oferta sea reseñable.

## Acceptance Criteria

- [ ] Backend: CRUD de `Commission` (crear / editar / listar / desactivar) con la oferta materia + término + docentes asignados.
- [ ] Frontend: pantalla de gestión de comisiones en el backoffice (`(staff)/admin`).
- [ ] La comisión enlaza `Subject` + `AcademicTerm` + `Teacher` (es la unidad reseñable).
