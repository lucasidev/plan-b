# UC-067: Crear/deshabilitar cuentas staff

**Actor**: Admin
**Epic**: [EPIC-09: Backoffice de cuentas staff](../epics/EPIC-09.md)
**User Story**: [US-067](../user-stories/US-067.md)
**ADR refs**: [ADR-0008](../../decisions/0008-roles-exclusivos-profiles-como-capacidades.md)

## Pre-condiciones

- Logueado como admin.

## Flujo principal

1. Admin crea User con `role IN ('moderator','admin','university_staff')` desde backoffice.
2. **No existe auto-registro para estos roles**.
3. Deshabilita seteando `disabled_at`.

## Post-condiciones

- Staff habilitado/deshabilitado.
