## Pre-condiciones

## Flujo principal

1. Visitante ingresa texto en el buscador.
2. Sistema hace full-text search sobre `Subject.name`, `Subject.code`, `Teacher.first_name`, `Teacher.last_name`.
3. Sistema ordena los resultados por relevancia.

## Post-condiciones

- Lista de matches con link a la página correspondiente.
