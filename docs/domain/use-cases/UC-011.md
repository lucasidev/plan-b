# UC-011: Verificar email

**Actor**: User recién registrado
**Epic**: [EPIC-02: Identidad y autenticación](../epics/EPIC-02.md)
**User Story**: [US-011-b](../user-stories/US-011-b.md), [US-011-f](../user-stories/US-011-f.md)
**ADR refs**: [ADR-0033](../../decisions/0033-verification-token-como-child-entity.md)

## Pre-condiciones

- Tiene un token de verificación válido recibido por email.

## Flujo principal

1. User hace click en el link recibido por email.
2. Sistema valida el token.
3. Sistema setea `email_verified_at=now()`.

## Post-condiciones

- Cuenta habilitada para crear profiles y publicar contenido.
