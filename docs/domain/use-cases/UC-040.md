# UC-040: Responder reseña sobre sí mismo

**Actor**: Docente verificado
**Epic**: [EPIC-06: Claim e identidad docente](../epics/EPIC-06.md)
**User Story**: [US-040](../user-stories/US-040.md)
**ADR refs**: [ADR-0008](../../decisions/0008-roles-exclusivos-profiles-como-capacidades.md)

## Pre-condiciones

- Existe una Review con `docente_reseñado_id` coincidente con su Teacher.
- No hay ya una `TeacherResponse` para esa reseña.

## Flujo principal

1. Docente abre la reseña.
2. Escribe `response_text`.
3. Sistema valida que es el docente_reseñado y que su TeacherProfile está verificado.
4. Sistema crea `TeacherResponse(status='published')`.

## Post-condiciones

- Respuesta visible públicamente bajo la reseña.
