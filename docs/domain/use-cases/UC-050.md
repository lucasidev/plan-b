# UC-050: Ver cola de reseñas under_review

**Actor**: Moderador
**Epic**: [EPIC-07 — Moderación](../epics/EPIC-07.md)
**User Story**: [US-050](../user-stories/US-050.md)
**ADR refs**: —

## Pre-condiciones

- Logueado como moderator o admin.

## Flujo principal

1. Moderador pide la cola.
2. Sistema devuelve Reviews con `status='under_review'` ordenadas por prioridad (cantidad de reports open, antigüedad).

## Post-condiciones

- Moderador puede atacar la cola.
