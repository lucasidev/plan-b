# UC-002: Ver página de una materia con reseñas

**Actor**: Visitante
**Epic**: [EPIC-01 — Catálogo público y exploración](../epics/EPIC-01.md)
**User Story**: [US-002](../user-stories/US-002.md)
**ADR refs**: [ADR-0009](../../decisions/0009-anonimato-como-regla-de-presentacion.md)

## Pre-condiciones

- La Subject existe en el catálogo.

## Flujo principal

1. Visitante pide la página de la Subject.
2. Sistema muestra metadata (código, año, horas, correlativas).
3. Sistema muestra agregados de reseñas (dificultad promedio, histograma, cantidad de reseñas).
4. Sistema muestra lista paginada de reseñas publicadas, sin identidad del autor.

## Post-condiciones

- Visitante lee reseñas de la materia.
