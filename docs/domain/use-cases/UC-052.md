# UC-052: Restaurar reseña removida

**Actor**: Moderador (típicamente ante apelación del autor)
**Epic**: [EPIC-07 — Moderación](../epics/EPIC-07.md)
**User Story**: [US-052](../user-stories/US-052.md)
**ADR refs**: —

## Pre-condiciones

- Review con `status='removed'`.

## Flujo principal

1. Moderador abre la reseña.
2. Justifica la restauración.
3. Sistema setea `Review.status='published'`.
4. Sistema registra `ReviewAuditLog` con `action='restored'`.

## Post-condiciones

- Reseña visible de nuevo.
- Los reports `upheld` históricos no se revierten.
