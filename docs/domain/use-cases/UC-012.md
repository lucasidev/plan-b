# UC-012: Crear StudentProfile eligiendo CareerPlan

**Actor**: Member con email verificado
**Epic**: [EPIC-02 — Identidad y autenticación](../epics/EPIC-02.md)
**User Story**: [US-012-b](../user-stories/US-012-b.md)
**ADR refs**: [ADR-0008](../../decisions/0008-roles-exclusivos-profiles-como-capacidades.md)

## Pre-condiciones

- Logueado.
- `email_verified_at NOT NULL`.
- No tiene un StudentProfile activo para esa carrera.

## Flujo principal

1. Member elige University.
2. Elige Career.
3. Elige CareerPlan (por default el vigente).
4. Indica `enrollment_year`.
5. Sistema crea `StudentProfile` con `status='active'`.

## Post-condiciones

- Member gana capacidades de alumno para esa carrera.
