# UC-018: Editar reseña propia

**Actor**: Alumno (autor de la reseña)
**Epic**: [EPIC-05 — Sistema de reseñas](../epics/EPIC-05.md)
**User Story**: [US-018](../user-stories/US-018.md)
**ADR refs**: [ADR-0005](../../decisions/0005-reseña-anclada-al-enrollment.md)

## Pre-condiciones

- Tiene Review propia con `status != 'removed'`.

## Flujo principal

1. Alumno abre la reseña.
2. Modifica los campos.
3. Sistema valida los cambios.
4. Sistema ejecuta el filtro automático sobre los nuevos textos (mismo criterio que UC-017).
5. Sistema actualiza `updated_at`.
6. Sistema registra `ReviewAuditLog` con `action='edited'` y `changes={before, after}`.

## Post-condiciones

- Reseña editada.
- UI muestra marca "editada" cuando `updated_at > created_at`.
- Si había `TeacherResponse`, el docente ve un badge "la reseña fue editada después de tu respuesta".
