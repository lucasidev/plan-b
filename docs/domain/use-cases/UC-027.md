# UC-027: Ver simulaciones públicas de otros (premium)

**Actor**: Alumno con StudentProfile activo + contribuciones
**Epic**: [EPIC-04: Planificación de cuatrimestre](../epics/EPIC-04.md)
**User Story**: [US-027](../user-stories/US-027.md)
**ADR refs**: [ADR-0028](../../decisions/0028-resenas-opcionales-y-premium-features-como-reward.md), [ADR-0029](../../decisions/0029-planning-bc-separado.md)

## Pre-condiciones

- Logueado.
- Ha publicado al menos una reseña.

## Flujo principal

1. Alumno pide listado vía `GET /api/simulations/public?planId=&termId=`.
2. Sistema retorna drafts shared del mismo plan/term, anonimizados, con composición + métricas agregadas.

## Post-condiciones

- Alumno tiene social proof / inspiración para sus propias decisiones.
