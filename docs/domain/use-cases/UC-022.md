# UC-022: Expirar registro no verificado (system action)

**Actor**: Sistema (scheduled job)
**Epic**: [EPIC-02 — Identidad y autenticación](../epics/EPIC-02.md)
**User Story**: [US-022](../user-stories/US-022.md)
**ADR refs**: —

## Pre-condiciones

- Existen Users con `email_verified_at=null` y `registered_at > 7 días atrás`.

## Flujo principal

1. Job daily corre.
2. Query identifica candidates.
3. Sistema setea `expired_at=now()` en cada uno.
4. Sistema emite event `UnverifiedRegistrationExpired`.

## Post-condiciones

- Email queda re-claimable (índice único parcial WHERE `expired_at IS NULL`).
- Audit log registrado.
