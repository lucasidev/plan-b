## Pre-condiciones

- Existe la University.

## Flujo principal

1. Admin crea Career.
2. Agrega CareerPlan asociado (version_label, duration_terms, default_term_kind, effective_from/to, notes).
3. Puede tener múltiples CareerPlans por Career a lo largo del tiempo.

## Post-condiciones

- Estructura lista para cargar materias.
