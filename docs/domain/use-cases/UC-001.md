# UC-001: Explorar catálogo de universidades y carreras

**Actor**: Visitante
**Epic**: [EPIC-01: Catálogo público y exploración](../epics/EPIC-01.md)
**User Story**: [US-001](../user-stories/US-001.md)
**ADR refs**: [ADR-0001](../../decisions/0001-multi-universidad-desde-dia-1.md), [ADR-0002](../../decisions/0002-versionado-de-planes-de-estudio.md)

## Pre-condiciones

- Ninguna.

## Flujo principal

1. Visitante navega el listado de Universities.
2. Elige una University y ve sus Careers.
3. Elige una Career y ve los CareerPlans (con el vigente destacado).
4. Elige un plan y ve las materias del plan con año, cuatrimestre y correlativas.

## Post-condiciones

- Visitante tiene contexto suficiente para buscar reseñas o registrarse.
