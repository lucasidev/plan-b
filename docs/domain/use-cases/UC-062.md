# UC-062: Gestionar Subject + Prerequisite

**Actor**: Admin
**Epic**: [EPIC-08: Backoffice de catálogo](../epics/EPIC-08.md)
**User Story**: [US-062](../user-stories/US-062.md)
**ADR refs**: [ADR-0003](../../decisions/0003-correlativas-con-dos-tipos.md)

## Pre-condiciones

- Existe el CareerPlan.

## Flujo principal

1. Admin hace CRUD sobre Subject dentro del plan.
2. Agrega correlativas `para_cursar` y `para_rendir` como rows de `Prerequisite`.
3. Sistema valida aciclicidad de cada grafo (por tipo) al insertar.

## Post-condiciones

- Plan de estudios cargado y listo para ser usado por alumnos.
