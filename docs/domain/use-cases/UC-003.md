# UC-003: Ver página de un docente con reseñas

**Actor**: Visitante
**Epic**: [EPIC-01 — Catálogo público y exploración](../epics/EPIC-01.md)
**User Story**: [US-003](../user-stories/US-003.md)
**ADR refs**: [ADR-0009](../../decisions/0009-anonimato-como-regla-de-presentacion.md)

## Pre-condiciones

- El Teacher existe en el catálogo.

## Flujo principal

1. Visitante pide la página del Teacher.
2. Sistema muestra metadata (nombre con title case, title, bio, universidad).
3. Sistema muestra agregados de reseñas donde fue `docente_reseñado`.
4. Sistema muestra lista paginada, sin identidad del autor.

## Post-condiciones

- Visitante lee reseñas del docente.
